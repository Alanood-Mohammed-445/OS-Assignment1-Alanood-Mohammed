## Question 1: What did you learn about multithreading?

**Your Answer:**
Multithreading is a programming technique that allows multiple threads to run concurrently within a single process.
It helps improve the performance and responsiveness of applications, especially when performing multiple tasks at the same time.
Each thread shares the same memory space but can execute independently.
However, multithreading can introduce challenges such as race conditions and deadlocks if not managed carefully.

## Question 4: How can you apply multithreading concepts in real-world applications?

**Your Answer:**
Multithreading can be applied in real-world applications to perform multiple tasks at the same time efficiently.
For example, it is used in web servers to handle multiple user requests simultaneously without slowing down.
It is also useful in applications like video streaming or gaming, where different processes (such as rendering and user input) run concurrently.
In mobile apps, multithreading helps keep the interface responsive while background tasks are executed.
Overall, it improves performance, responsiveness, and better utilization of system resources.
